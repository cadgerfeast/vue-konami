## Installation

``` bash
npm install vue-konami --save
```

## Usage

### Simple

``` html
<div v-konami="easterEgg"></div>

<script setup>
  const easterEgg = () => {
    alert('easterEgg');
  };
</script>
```

### Advanced

``` html
<div v-konami="easterEgg"></div>

<script setup>
  const easterEgg = {
    timeout: 500,
    chain: ['A', 'Z', 'E', 'R', 'T', 'Y'],
    callback: () => {
      alert('easterEgg');
    }
  };
</script>
```
